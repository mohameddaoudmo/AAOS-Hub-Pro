INTRODUCTION
ما هو Car UI Library؟
الفرق بين Android العادي وتطبيقات السيارة

Car UI Library (car-ui-lib) هو نظام مكونات خاص بـ Android Automotive OS — ليس مثل Material Design العادي. يوفر مكونات محسّنة للقيادة: كبيرة الحجم، ملائمة للمس باليد، ومتوافقة مع Driver Distraction Guidelines.

📱 Android عادي
  • Material Design قياسي
  • أي حجم عناصر
  • مناسب للأصابع الدقيقة
  • لا قيود على الانتباه
🚗 Car App Library (Auto)
  • Templates فقط — لا layouts حرة
  • عناصر كبيرة جداً للقيادة
  • يعمل على هاتف عبر Android Auto
  • Driver Distraction مدمج
🖥️ Car UI Lib (AAOS)
  • مكونات OEM-customizable
  • مدمج في النظام مباشرة
  • يعمل على شاشة السيارة
  • OEM يكيّف المظهر
⚡
الفرق الجوهري — Car App Library vs Car UI Library
  • Car App Library (androidx.car.app): للمطورين الخارجيين — يعمل على Android Auto (projection) وAAOS
  • Car UI Library (car-ui-lib): للـ OEM وتطبيقات النظام — مكونات AAOS المدمجة مثل Toolbar، RecyclerView، Preferences
  • هذا الدليل يشمل الاثنين بالتفصيل

SETUP
الإعداد من الصفر
Dependencies، Manifest، ومتطلبات البناء
1

أنشئ مشروع Android جديد

في Android Studio: New Project → Automotive → No Activity. أو أضف module للمشروع الموجود.

2

أضف Dependencies

في build.gradle أضف Car App Library واضبط minSdk على 23 على الأقل.

3

اضبط AndroidManifest

أعلن عن CarAppService وحدد Category التطبيق (Navigation، Media، Parking، إلخ).

4

ثبّت Desktop Head Unit

من Android Studio SDK Manager حمّل DHU لاختبار التطبيق على محاكي شاشة السيارة.

// build.gradle — Module level

class MyCarAppService : CarAppService() {

    // تحديد من يحق له الاتصال
    override fun createHostValidator(): HostValidator {
        return if (BuildConfig.DEBUG) {
            HostValidator.ALLOW_ALL_HOSTS_VALIDATOR
        } else {
            HostValidator.Builder(applicationContext)
                .addAllowedHosts(R.array.allowed_car_hosts)
                .build()
        }
    }

    // إنشاء Session جديدة لكل اتصال
    override fun onCreateSession(): Session = MySession()
}

// ═══════════════════════════════
// 2. Session — إدارة دورة الحياة
// ═══════════════════════════════
class MySession : Session() {

    // الشاشة الأولى عند فتح التطبيق
    override fun onCreateScreen(intent: Intent): Screen {
        // معالجة Deep Links
        val action = intent.action
        return when (action) {
            CarContext.ACTION_NAVIGATE -> NavigationScreen(carContext, intent.data)
            "com.example.SHOW_DETAIL"    -> DetailScreen(carContext)
            else                         -> HomeScreen(carContext)
        }
    }

    // لما يوصل intent جديد (deep link)
    override fun onNewIntent(intent: Intent) {
        val screen = screenManager.top
        if (screen is HomeScreen) {
            screen.handleNewIntent(intent)
        }
    }
}

// 3. Screen — وحدة العرض

class HomeScreen(carContext: CarContext) : Screen(carContext) {

    // State الشاشة
    private var isLoading = true
    private var items: List<PlaceData> = emptyList()

    init {
        // تحميل البيانات في background
        lifecycleScope.launch {
            items = loadPlaces()
            isLoading = false
            invalidate()  // أعد رسم الشاشة
        }
    }

    // ⭐ يُستدعى كلما تغيّر الـ state
    override fun onGetTemplate(): Template {
        return if (isLoading) {
            buildLoadingTemplate()
        } else {
            buildMainTemplate()
        }
    }

    private fun buildLoadingTemplate(): Template {
        return MessageTemplate.Builder("جاري التحميل...")
            .setLoading(true)
            .build()
    }

    private fun buildMainTemplate(): Template {
        return ListTemplate.Builder()
            .setTitle("الرئيسية")
            .setSingleList(buildItemList())
            .setHeaderAction(Action.APP_ICON)
            .build()
    }
}
🔄
دورة حياة Screen
  • onGetTemplate() — يُستدعى عند الحاجة لرسم الشاشة
  • invalidate() — تطلب من النظام استدعاء onGetTemplate() مجدداً
  • screenManager.push() — فتح شاشة جديدة (يضيفها للـ stack)
  • screenManager.pop() — الرجوع للشاشة السابقة
  • screenManager.popTo() — الرجوع لشاشة محددة في الـ stack

class PlacesScreen(carContext: CarContext) : Screen(carContext) {

    override fun onGetTemplate(): Template {
        return ListTemplate.Builder()
            .setTitle("الأماكن القريبة")

            // Header action (back أو app icon)
            .setHeaderAction(Action.BACK)

            // Action Strip في الـ header
            .setActionStrip(
                ActionStrip.Builder()
                    .addAction(
                        Action.Builder()
                            .setTitle("تصفية")
                            .setOnClickListener { showFilterScreen() }
                            .build()
                    )
                    .build()
            )

            // قائمة واحدة
            .setSingleList(buildSingleList())

            // أو: قوائم متعددة بعناوين (Sectioned)
            // .addSectionedList(buildSection1(), "وقود")
            // .addSectionedList(buildSection2(), "شحن")

            .build()
    }

    private fun buildSingleList(): ItemList {
        val places = getPlaces()  // List<Place>

        return ItemList.Builder()
            // Row بسيط
            .addItem(
                Row.Builder()
                    .setTitle("محطة وقود Shell")

                    // نصوص إضافية (max 2)
                    .addText("1.2 km · مفتوح 24 ساعة")
                    .addText("4.2 ★ · بنزين 92, 95")

                    // أيقونة
                    .setImage(
                        CarIcon.Builder(
                            IconCompat.createWithResource(carContext, R.drawable.ic_fuel)
                        ).setTint(CarColor.GREEN).build()
                    )

                    // Toggle (Radio button)
                    // .setToggle(Toggle.Builder { isChecked -> }.build())

                    // Click handler
                    .setOnClickListener {
                        screenManager.push(PlaceDetailScreen(carContext, "shell"))
                    }
                    .build()
            )
            // Row بـ Decoration (Badge)
            .addItem(
                Row.Builder()
                    .setTitle("شاحن Tesla")
                    .addText("3.1 km · 8 منافذ متاحة")
                    .setImage(
                        CarIcon.Builder(
                            IconCompat.createWithResource(carContext, R.drawable.ic_ev)

                                .setColor(CarColor.RED)

            .build()

class MainMenuScreen(carContext: CarContext) : Screen(carContext) {

    override fun onGetTemplate(): Template {
        return GridTemplate.Builder()
            .setTitle("القائمة الرئيسية")
            .setHeaderAction(Action.APP_ICON)
            .setSingleList(
                ItemList.Builder()
                    .addItem(buildGridItem("ملاحة",    R.drawable.ic_map)       { goNav() })
                    .addItem(buildGridItem("موسيقى",   R.drawable.ic_music)     { goMusic() })
                    .addItem(buildGridItem("هاتف",     R.drawable.ic_phone)     { goPhone() })
                    .addItem(buildGridItem("مناخ",     R.drawable.ic_climate)   { goClimate() })
                    .addItem(buildGridItem("إعدادات",  R.drawable.ic_settings)  { goSettings() })
                    .addItem(buildGridItem("بطارية",   R.drawable.ic_battery)   { goBattery() })
                    .build()
            )
            .build()
    }

    private fun buildGridItem(
        title: String,
        iconRes: Int,
        onClick: () -> Unit
    ): GridItem {
        return GridItem.Builder()
            .setTitle(title)
            .setImage(
                CarIcon.Builder(
                    IconCompat.createWithResource(carContext, iconRes)
                ).build()
            )
            .setOnClickListener(onClick)
            .build()
    }
}

class ActiveNavScreen(carContext: CarContext) : Screen(carContext) {

    private var currentStep: Step? = null
    private var eta: TravelEstimate? = null

    override fun onGetTemplate(): Template {
        return NavigationTemplate.Builder()

            // ═══ معلومات المسار الحالي ═══
            .setNavigationInfo(
                RoutingInfo.Builder()

                    // الخطوة الحالية
                    .setCurrentStep(
                        Step.Builder("انعطف يميناً على شارع النيل")
                            .setManeuver(
                                Maneuver.Builder(Maneuver.TYPE_TURN_NORMAL_RIGHT)
                                    // أيقونة مخصصة للمنعطف
                                    .setIcon(CarIcon.Builder(
                                        IconCompat.createWithResource(carContext, R.drawable.turn_right)
                                    ).build())
                                    .build()
                            )
                            .setRoad("شارع النيل")
                            // معلومات الحارات
                            .addLane(
                                Lane.Builder()
                                    .addDirection(LaneDirection.create(
                                        LaneDirection.SHAPE_NORMAL_RIGHT, true
                                    ))
                                    .build()
                            )
                            .build()
                    )

                    // المسافة للخطوة التالية
                    .setCurrentDistance(
                        Distance.create(500.0, Distance.UNIT_METERS)
                    )

                    // الخطوة التالية (اختياري)
                    .setNextStep(
                        Step.Builder("استمر على الطريق الدائري")
                            .build()
                    )
                    .build()
            )

            // ═══ وقت الوصول التقديري ═══
            .setDestinationTravelEstimate(
                TravelEstimate.Builder(
                    Distance.create(15.2, Distance.UNIT_KILOMETERS),
                    ZonedDateTime.now().plusMinutes(22)
                )
                .setRemainingTimeSeconds(1320)  // 22 دقيقة بالثواني
                .setRemainingTimeColor(CarColor.GREEN)
                .setRemainingDistanceColor(CarColor.YELLOW)
                .build()
            )

            // ═══ أزرار الخريطة ═══
            .setMapActionStrip(
                ActionStrip.Builder()
                    .addAction(
                        Action.Builder()
                            .setIcon(CarIcon.Builder(
                                IconCompat.createWithResource(carContext, R.drawable.ic_recenter)
                            ).build())
                            .setOnClickListener { recenterMap() }
                            .build()
                    )
                    .addAction(buildZoomInAction())
                    .addAction(buildZoomOutAction())
                    .build()
            )

            // ═══ ActionStrip السفلي ═══
            .setActionStrip(
                ActionStrip.Builder()
                    .addAction(
                        Action.Builder()
                            .setTitle("وصلت")
                            .setBackgroundColor(CarColor.GREEN)
                            .setOnClickListener { onArrived() }
                            .build()
                    )
                    .addAction(
                        Action.Builder()
                            .setTitle("توقف")
                            .setOnClickListener { stopNavigation() }
                            .build()
                    )
                    .build()
            )

            // لرسم الخريطة: SurfaceCallback
            .setPanModeListener { isPanMode ->
                onPanModeChanged(isPanMode)
            }

            .build()
    }
}

@Suppress("DEPRECATION")
class CarMusicService : MediaBrowserServiceCompat() {

    private lateinit var mediaSession: MediaSessionCompat
    private lateinit var audioManager: AudioManager
    private lateinit var audioFocusRequest: AudioFocusRequest

    override fun onCreate() {
        super.onCreate()
        audioManager = getSystemService(AudioManager::class.java)

        // إنشاء MediaSession
        mediaSession = MediaSessionCompat(this, "CarMusic").apply {
            setFlags(
                MediaSessionCompat.FLAG_HANDLES_MEDIA_BUTTONS or
                MediaSessionCompat.FLAG_HANDLES_TRANSPORT_CONTROLS
            )
            setCallback(mediaCallback)
            isActive = true
        }

        sessionToken = mediaSession.sessionToken
    }

    // ═══ Media Callbacks — أزرار التحكم ═══
    private val mediaCallback = object : MediaSessionCompat.Callback() {

        override fun onPlay() {
            // طلب AudioFocus أولاً
            audioFocusRequest = AudioFocusRequest.Builder(AudioManager.AUDIOFOCUS_GAIN)
                .setOnAudioFocusChangeListener { focus ->
                    when (focus) {
                        AudioManager.AUDIOFOCUS_LOSS          -> pause()
                        AudioManager.AUDIOFOCUS_LOSS_TRANSIENT -> pause()
                        AudioManager.AUDIOFOCUS_GAIN          -> resume()
                    }
                }
                .build()

            val result = audioManager.requestAudioFocus(audioFocusRequest)
            if (result == AudioManager.AUDIOFOCUS_REQUEST_GRANTED) {
                startPlayback()
                mediaSession.isActive = true
                updateState(PlaybackStateCompat.STATE_PLAYING)
                startForeground(1, buildNotification())
            }
        }

        override fun onPause() {
            pausePlayback()
            updateState(PlaybackStateCompat.STATE_PAUSED)
            audioManager.abandonAudioFocusRequest(audioFocusRequest)
        }

        override fun onSkipToNext()     { playNext(); updateMetadata() }
        override fun onSkipToPrevious() { playPrev(); updateMetadata() }
        override fun onStop()            { stopPlayback(); stopForeground(true) }

        override fun onPlayFromMediaId(mediaId: String?, extras: Bundle?) {
            mediaId?.let { playTrack(it) }
        }

        // Custom Actions (زر إضافي في السيارة)
        override fun onCustomAction(action: String?, extras: Bundle?) {
            when (action) {
                "SHUFFLE" -> toggleShuffle()
                "REPEAT"  -> toggleRepeat()
            }
        }
    }

    // ═══ PlaybackState — يُحدَّث باستمرار ═══
    private fun updateState(state: Int) {
        val playbackState = PlaybackStateCompat.Builder()
            .setState(state, currentPosition, 1.0f)
            .setActions(
                PlaybackStateCompat.ACTION_PLAY                  or
                PlaybackStateCompat.ACTION_PAUSE                 or
                PlaybackStateCompat.ACTION_SKIP_TO_NEXT          or
                PlaybackStateCompat.ACTION_SKIP_TO_PREVIOUS     or
                PlaybackStateCompat.ACTION_PLAY_FROM_MEDIA_ID
            )
            // Custom actions تظهر في السيارة
            .addCustomAction(
                PlaybackStateCompat.CustomAction.Builder(
                    "SHUFFLE",
                    "خلط عشوائي",
                    R.drawable.ic_shuffle
                ).build()
            )
            .build()

        mediaSession.setPlaybackState(playbackState)
    }

    // ═══ Metadata — معلومات الأغنية ═══
    fun updateMetadata(track: Track) {
        mediaSession.setMetadata(
            MediaMetadataCompat.Builder()
                .putString(METADATA_KEY_TITLE,  track.title)
                .putString(METADATA_KEY_ARTIST, track.artist)
                .putString(METADATA_KEY_ALBUM,  track.album)
                .putBitmap(METADATA_KEY_ALBUM_ART, track.artwork)
                .putLong(METADATA_KEY_DURATION,   track.durationMs)
                .build()
        )
    }

    // ═══ MediaBrowser Tree ═══
    override fun onGetRoot(pkg: String, uid: Int, hints: Bundle?): BrowserRoot {
        // تحقق أن المتصل هو Android Auto / AAOS
        return BrowserRoot("/root", null)
    }

    override fun onLoadChildren(parentId: String, result: Result<MutableList<MediaItem>>) {
        result.detach()  // async loading

        lifecycleScope.launch {
            val items = when (parentId) {
                "/root"     -> getRootItems()    // قوائم التشغيل, الأحدث
                "/recent"  -> getRecentTracks()
                "/albums"  -> getAlbums()
                else       -> getTracksForId(parentId)
            }
            result.sendResult(items.toMutableList())
        }
    }
}

🛠️ UI Implementation Blueprints

أنماط هندسية لضمان استقرار واجهة المستخدم في بيئة السيارة:

  • State-Driven Invalidation: استخدام invalidate() فقط عند تغير البيانات الفعلية.
  • Template Quota Management: تتبع الـ stack size لمنع تجاوز حد الـ 5 templates.
  • Lifecycle-Aware MediaSession: ربط الـ Session بـ LifecycleService لضمان تحرير الموارد.
⚡ Optimization Signals

علامات تحذيرية يجب مراقبتها أثناء تطوير الـ Car UI:

  • UI Lag during Driving: مؤشر على ثقل الـ onGetTemplate أو عمليات blocking.
  • ANR on Start: غالباً بسبب تأخر الـ MediaBrowser connection في الـ main thread.
  • Template Rejection: يحدث عند محاولة إرسال Template غير مدعوم للـ Host الحالي.
