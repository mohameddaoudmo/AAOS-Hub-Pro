01

ما هي Linked List؟

🔗

Node

وحدة البناء الأساسية. كل Node بتحتوي على data (القيمة) وpointer بيشير للـ node الجاية.

🏠

Head

أول node في القائمة. ده بوابة دخول القائمة كلها. لو فقدته، فقدت القائمة كلها!

🛑

Tail

آخر node في القائمة. الـ next pointer بتاعه بيشير لـ NULL (نهاية القائمة).

📦

Dynamic Size

مش زي Arrays — مش محتاج تعرف الحجم مسبقاً. بتكبر وتصغر ديناميكياً وقت التشغيل.

مقارنة Array vs Linked List
العملية Array Linked List ملاحظة
Access by index O(1) O(n) Array أسرع
Insert at beginning O(n) O(1) Linked List أسرع
Insert at end O(1) amortized O(1) لو عندك tail متشابهين
Delete from middle O(n) O(1) لو عندك pointer Linked List أسرع
Search O(n) O(n) متشابهين

Singly Linked List — القائمة أحادية الاتجاه

كل node عندها pointer واحد بس بيشير للـ node الجاية. يعني مقدرش أرجع للخلف!
الحركة فيها في اتجاه واحد فقط من اليسار لليمين.

🎬 أنيميشن Singly Linked List — إضافة وحذف
Insert at Head — إضافة في البداية O(1)
Python 
def insert_at_head(self, data):

    new_node = Node(data)

    new_node.next = self.head  # ①  الـ node الجديد يشير للـ head القديم

    self.head = new_node       # ②  الـ head يبقى الـ node الجديد

    # ✅ O(1) — سريع جداً
Insert at Tail — إضافة في النهاية O(n)
Python 
def insert_at_tail(self, data):

    new_node = Node(data)

    if not self.head:

        self.head = new_node

        return

    

    curr = self.head

    while curr.next:   # ① امشي لحد ما توصل للأخير

        curr = curr.next

    curr.next = new_node  # ② اربط الـ new node

    # ⚠️ O(n) — لو عندك tail pointer تبقى O(1)
Delete by Value — حذف بالقيمة O(n)
Python 
def delete(self, val):

    if not self.head: return

    

    # Case 1: الـ head هو اللي هنحذفه

    if self.head.data == val:

        self.head = self.head.next

        return

    

    # Case 2: ابحث عن الـ node قبل المطلوب

    prev = self.head

    while prev.next and prev.next.data != val:

        prev = prev.next

    

    if prev.next:  # وجدناه

        prev.next = prev.next.next  # ① "قفز" فوق الـ node المحذوف
💡
تريك مهم: لو محتاج تحذف node وعندك pointer عليها مباشرة (مش head)، ممكن تنسخ القيمة من الـ node الجاية وتحذف الـ node الجاية بدل ما تتعب تجيب الـ prev!

Doubly Linked List — القائمة ثنائية الاتجاه

كل node عندها pointerين: next للأمام وprev للخلف.
ممكن أتحرك في الاتجاهين! ده بيفيد جداً في عمليات معينة.

🎬 أنيميشن Doubly Linked List — حركة في الاتجاهين
Node وكلاس الـ Doubly List
Python 
class DoublyNode:

    def __init__(self, data):

        self.data = data

        self.next = None    # ← الأمام

        self.prev = None    # ← الخلف



def insert_at_tail(self, data):

    new_node = DoublyNode(data)

    if not self.head:

        self.head = self.tail = new_node

        return

    # ① ربط الـ new node

    new_node.prev = self.tail

    self.tail.next = new_node

    self.tail = new_node

    # ✅ O(1) — لأن عندنا tail pointer!



def delete_node(self, node):

    # ✨ مش محتاجين ندور على الـ prev!

    if node.prev: node.prev.next = node.next

    else: self.head = node.next  # كان head

    if node.next: node.next.prev = node.prev

    else: self.tail = node.prev  # كان tail

    # ✅ O(1) — مزية الـ doubly!
✅

مزايا Doubly

حذف O(1) لو عندك pointer
traversal في الاتجاهين
مناسب للـ LRU Cache
سهل عكس القائمة

⚠️

عيوب Doubly

ذاكرة أكثر (pointer زيادة)
كود أعقد
كل إضافة/حذف لازم تعدل pointerين

⚠️
خطأ شائع جداً: لما تحذف node في doubly، متنساش تعدل الـ prev pointer بتاع الـ node الجاية والـ next pointer بتاع الـ node السابقة! لو نسيت واحد فيهم هتعمل corruption في القائمة.

Circular Linked List — القائمة الدائرية

آخر node مش بتشير لـ NULL، بتشير للـ head مرة تانية!
يعني القائمة زي حلقة — مفيش بداية ومفيش نهاية.

🎬 أنيميشن Circular Linked List
↑ الـ tail.next بيرجع للـ head ↑
كود Circular List — Insert & Traverse
Python 
def insert(self, data):

    new_node = Node(data)

    if not self.head:

        self.head = new_node

        new_node.next = self.head  # يشير لنفسه!

        return

    # ابحث عن الـ tail

    tail = self.head

    while tail.next != self.head:

        tail = tail.next

    tail.next = new_node

    new_node.next = self.head  # أهم سطر! الربط الدائري



def traverse(self):

    if not self.head: return

    curr = self.head

    while True:

        print(curr.data)

        curr = curr.next

        if curr == self.head:  # ⚠️ الشرط: رجعنا للبداية

            break

Reversal — قلب الـ Linked List

من أكتر المسائل اللي بتيجي في الإنترفيوز! محتاج تعكس اتجاه كل الـ pointers.

🎬 أنيميشن Reversal — خطوة بخطوة
الخوارزمية Iterative — 3 pointers
1

بنعمل 3 variables: prev = None, curr = head, next_node = None

2

في كل iteration: بنحفظ next_node = curr.next (عشان مانضيعش باقي القائمة)

3

بنعكس الـ pointer: curr.next = prev (القلب الفعلي)

4

بنحرك الـ pointers للأمام: prev = curr, curr = next_node

5

لما curr == None، الـ prev بيبقى الـ head الجديد!

Python — Iterative O(n) time, O(1) space 
def reverse(self):

    prev = None

    curr = self.head

    

    while curr:

        next_node = curr.next  # ① احفظ الـ next

        curr.next = prev        # ② اقلب الـ pointer

        prev = curr             # ③ حرك prev للأمام

        curr = next_node        # ④ حرك curr للأمام

    

    self.head = prev  # ⑤ prev الجديد = head الجديد

# Recursive (elegant, O(n) space)

# ════════════════════════════════

def reverse_recursive(self, node):

    if not node or not node.next:

        return node  # Base case: آخر node هو الـ head الجديد

    

    new_head = self.reverse_recursive(node.next)

    node.next.next = node  # اقلب الـ pointer

    node.next = None        # ازل الـ old pointer

    return new_head
Reverse a Sublist — عكس جزء فقط (مسألة LeetCode 92)
Python — Reverse from position left to right 
def reverseBetween(head, left, right):

    dummy = Node(0)

    dummy.next = head

    prev = dummy

    

    # ① امشي لـ (left - 1)

    for _ in range(left - 1):

        prev = prev.next

    

    curr = prev.next

    

    # ② اعكس (right - left) مرات

    for _ in range(right - left):

        next_node = curr.next

        curr.next = next_node.next

        next_node.next = prev.next

        prev.next = next_node

    

    return dummy.next
Time: O(n)
Space (iterative): O(1)
Space (recursive): O(n)

Cycle Detection — اكتشاف الحلقة 🐢🐰

أحياناً في قائمة، آخر node بيشير لـ node في المنتصف بدل NULL — فبيتكون cycle (حلقة).
الخوارزمية الأشهر لاكتشافها هي Floyd's Algorithm أو "السلحفاة والأرنب" 🐢🐰.

🎬 Floyd's Cycle Detection — الـ Slow و Fast Pointers
Floyd's Algorithm — كود كامل
Python — Detect, Find Entry, Find Length 
# ═══════════════════════════════════

# ① هل فيه Cycle؟ — O(n) T, O(1) S

# ═══════════════════════════════════

def has_cycle(head):

    slow = fast = head

    while fast and fast.next:

        slow = slow.next         # 🐢 خطوة واحدة

        fast = fast.next.next    # 🐰 خطوتين

        if slow == fast:

            return True          # التقوا = فيه cycle

    return False



# ═══════════════════════════════════════════

# ② إيه نقطة بداية الـ Cycle؟

# ═══════════════════════════════════════════

# بعد التقاء slow و fast:

# - ارجع slow للـ head

# - حرك الاتنين خطوة خطوة

# - مكان التقاؤهم = بداية الـ cycle ✨

def find_cycle_entry(head):

    slow = fast = head

    while fast and fast.next:

        slow = slow.next

        fast = fast.next.next

        if slow == fast:

            slow = head         # ← أعد slow للـ head

            while slow != fast:

                slow = slow.next

                fast = fast.next

            return slow        # بداية الـ cycle!

    return None



# ════════════════════════════

# ③ طول الـ Cycle؟

# ════════════════════════════

def cycle_length(head):

    slow = fast = head

    while fast and fast.next:

        slow = slow.next

        fast = fast.next.next

        if slow == fast:           # التقوا

            count = 1

            fast = fast.next

            while slow != fast:   # دور على نفسك وعد

                fast = fast.next

                count += 1

            return count

    return 0
ليه الـ Floyd Algorithm شغال؟ — البرهان الرياضي

تخيل الـ cycle بدأ بعد F خطوات من الـ head، وطوله C.
لما slow يوصل لبداية الـ cycle، fast كان مشى 2F خطوة — يعني مشى F steps جوه الـ cycle.
المسافة بين fast وبداية الـ cycle = C - (F mod C).
لما تحرك slow من الـ head وfast من مكان الالتقاء بنفس السرعة، بيلتقوا عند بداية الـ cycle بالظبط! ✨

Time: O(n)
Space: O(1)
الاسم: Floyd / Tortoise & Hare

# إيجاد المنتصف (Middle of Linked List)

# ════════════════════════════════════

def find_middle(head):

    slow = fast = head

    while fast and fast.next:

        slow = slow.next

        fast = fast.next.next

    return slow  # ← المنتصف! 🎯



# ═══════════════════════════════════════════════════

# إيجاد الـ N-th node from the END

# فكرة: fast يمشي n خطوة أول، تعد يمشوا مع بعض

# ═══════════════════════════════════════════════════

def find_nth_from_end(head, n):

    fast = slow = head

    for _ in range(n):    # fast يمشي n خطوة

        fast = fast.next

    while fast:            # امشوا مع بعض

        slow = slow.next

        fast = fast.next

    return slow            # ← الـ N-th from end!



# ════════════════════════════════════════

# Intersection of Two Lists

# ════════════════════════════════════════

# len(A) + len(B) = len(B) + len(A) ✨

def get_intersection(headA, headB):

    a, b = headA, headB

    while a != b:

        a = a.next if a else headB

        b = b.next if b else headA

    return a  # نقطة التقاطع أو None



# ══════════════════════════════════

# Merge Two Sorted Lists

# ══════════════════════════════════

def merge_sorted(l1, l2):

    dummy = Node(0)  # Dummy Node!

    curr = dummy

    while l1 and l2:

        if l1.data <= l2.data:

            curr.next = l1; l1 = l1.next

        else:

            curr.next = l2; l2 = l2.next

        curr = curr.next

    curr.next = l1 or l2

    return dummy.next

Reverse K-Group — الكود الكامل
Python — LeetCode #25 Hard 
def reverseKGroup(head, k):

    # ① تحقق إن فيه k nodes

    count, curr = 0, head

    while curr and count < k:

        curr = curr.next

        count += 1

    if count < k: return head  # أقل من k nodes، رجع كما هو

    

    # ② اعكس الـ k nodes

    prev, curr = None, head

    for _ in range(k):

        next_node = curr.next

        curr.next = prev

        prev = curr

        curr = next_node

    

    # ③ head الأصلي بقى tail — اربطه بالباقي

    head.next = reverseKGroup(curr, k)

    return prev  # prev = head الجديد للـ group
